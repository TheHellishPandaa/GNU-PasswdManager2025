GNU-PasswdManager

Manage your passwords with encryption and a user-friendly GUI. Allows users to store, generate, and manage their passwords securely.

©2025 Jaime Galvez Martinez

This project is released under GNU License

Download the Latest Version

GNU Passwd Manager is a tool that helps you manage your passwords securely using encryption. You can easily store and retrieve your passwords with a simple graphical interface.

Choose the version for your operating system:

For Linux/Mac:

Download .tar.gz Package for Linux

After downloading the package, you can install it on your Linux system using the following command:

sudo tar -xzvf GNU-PasswdManager-Lastest.tar.gz

If you encounter any issues with dependencies, you can read the manual (readme) from GitHub:
