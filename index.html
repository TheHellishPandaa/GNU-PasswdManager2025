GNU Passwd Manager

Securely manage your passwords with encryption and a user-friendly GUI

GNU Passwd Manager is a tool that helps you manage your passwords securely using encryption. You can easily store and retrieve your passwords with a simple graphical interface.

Download .tar.gz Package for Linux

sudo tar -xzvf GNU-PasswdManager-Lastest.tar.gz

If you encounter any issues with dependencies, you can fix them by running:

sudo apt-get install -f

Download .exe Installer for Windows

For Windows, simply run the installer after downloading it to your system.
